# Review Sentiment Analysis Using RNN
This project focuses on the sentiment analysis of text reviews using RNN . It analyzes customer reviews and predict sentiment based on trained models.
Access to the portal is restricted using user authentication. for access user have to first signup to the portal.
This project was develped as a **learning project** , and predictions may not always be accurate.

# features
- user authentication
- sentiment predictions using RNN
- support for short and long text reviews
- separate models for different review lengths
- Simple and user-friendlt interface

# technologies used
- python
- tensorflow
- pandas
- numpy
- streamlit

# Important note
this is a learning project
models are trained on limited datasets.
predictions may sometimes be incorrect
this project for educational and practice purposes only
